# Installation
Install required python packages *pandas* and *Flask*
```bash
pip install pandas
pip install Flask
```

# Run
To run server execute scritp

## Windows
```cmd
run.bat
```

## Linux/Mac
First time, make script executable (**Only first time**)
```bash
chmod +x run.sh
```
Then execute
```bash
./run.sh
```
